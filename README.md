# frontend-assignment

## Requirements
- Install Node.js

## How to use

### Backend
Simple dummy backend API is implemented.

```
$ cd backend
$ npm install
$ node index.js
```

### Frontend
```
$ cd frontend
$ npm install
$ npm run build
$ npm run start
```

### Usage
1. open `http://localhost:3000`
1. input inbox ID and click "メールを開く". As backend mock, 123456 and 234567 are only available.
1. mock emails will be listed.
